animate

animate()
Creates a new animation for the target object.
Note: Animations are short-lived and removed once completed.
Parameters
Name Type Required Description
target object|Array|NodeList|string|function Yes The target object to animate. Optionally a selector, function or array.
properties object|Array Yes The properties to animate, either as an object or entries ([[property, [from, to]]]).
properties.property object No The property to animate. Value can be a number or instance of LS.Color. Optionally you can set an initial value as [from, to].
options object Yes The animation options
options.easing string Yes The easing function to use (linear, ease-in, ease-out, ease-in-out)
options.duration number Yes The duration of the animation in milliseconds
options.cutGroup number Yes The cut group ID. Animating with this ID stops any other animations with the same ID.
group string Yes The animation group

static fadeOut()
Helper method for quick fade out, backwards compatibile with Animation 1.x; animates opacity to 0 and moves in a direction.
Examples
LS.Animation.fadeOut(element, { duration: 500, direction: 'down' }).then(() => { console.log("Faded out!"); });
