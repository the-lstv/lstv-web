Adds a resize handle to a target element. It can be called multiple times on the same element to change sides or options (upsert).

Certain options (anchor, size) can also be set per specific handle with CSS (eg. element > .ls-resize-handle.ls-top { --ls-resize-handle-size: 8px; --ls-resize-anchor: 0.5 })
Parameters
Name Type Required Default Description
target * Yes The target element to resize.
options * Yes The options for the resize handle.

Sides and corners can be specified in four ways:
- As a CSS-style array of booleans/numbers (e.g. { sides: [true, false, true, false] }), clockwise
- As an array of strings (e.g. { sides: ["top", "bottom"] })
- As keys { top: true, right: false, bottom: true, left: false }
- All at once { sides: "all" }

Other options:
options.styled boolean No true Whether to apply default visual styles, eg. highlight on hover. Otherwise only functional styles will apply.
options.sides boolean|Array|string No true Which sides to add handles to. Can be a boolean (true = all sides), an array of booleans/numbers [top, right, bottom, left], or an array of strings ["top", "right", "bottom", "left"].
options.corners boolean|Array|string No false Which corners to add handles to. Can be a boolean (true = all corners), an array of booleans/numbers [top-left, top-right, bottom-right, bottom-left], or an array of strings ["top-left", "top-right", "bottom-right", "bottom-left"].
options.top boolean No Whether to add a handle to the top side.
options.right boolean No Whether to add a handle to the right side.
options.bottom boolean No Whether to add a handle to the bottom side.
options.left boolean No Whether to add a handle to the left side.
options.topLeft boolean No Whether to add a handle to the top-left corner.
options.topRight boolean No Whether to add a handle to the top-right corner.
options.bottomRight boolean No Whether to add a handle to the bottom-right corner.
options.bottomLeft boolean No Whether to add a handle to the bottom-left corner.
options.handleSize number|string No Size of the side handles (thickness).
options.cornerSize number|string No Size of the corner handles (square size).
options.anchor string No 0.5 Whether handles should be outside, inside, or in the center of the edge. (0 = inside, 0.5 = center, 1 = outside)
options.cursors boolean No true Whether to set the cursor when dragging (this does not effect the CSS cursor style of the handle itself).
options.boundsCursors boolean No true Set the cursor to a single direction when dragging past min/max.
options.snapArea number No 40 Pixel threshold for snapping.
options.snapCollapse boolean No false If true, shrinking within snapArea snaps to collapsed height/width.
options.snapExpand boolean No false If true, expanding within snapArea of parent height snaps to 100% height/width.
options.snapVertical boolean No false If true, enables snapping vertically.
options.snapHorizontal boolean No false If true, enables snapping horizontally.
options.minWidth number No 20 Minimum width of the target element (overrides CSS min-width value).
options.minHeight number No 20 Minimum height of the target element (overrides CSS min-height value).
options.maxWidth number No null Maximum width of the target element (overrides CSS max-width value).
options.maxHeight number No null Maximum height of the target element (overrides CSS max-height value).
options.boundary string|object No null Boundary to constrain resizing. Can be "viewport" or a rect object {x, y, width, height}.
options.store string No null Key name for storage. If set, updates will be persistent and saved into a storage object.
options.storeStringify boolean No true Whether to stringify the stored data.
options.storage object No null Custom storage object (must implement getItem/setItem). Default is localStorage.
options.translate boolean No Use translate3d instead of left/top
Returns
- value: An object with the registered handles
Events on handle:
- resize: Emitted when the element is resized with the new width, height, and state.
- start: Emitted when resizing starts.
- end: Emitted when resizing ends.
Examples
LS.Resize.set(element, {
    sides: ["top", "bottom"],
    corners: [1, 0, 1, 0]
});
