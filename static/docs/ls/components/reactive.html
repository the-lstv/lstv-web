wrap

wrap()
Wraps an object with a reactive proxy.
The proxy will get the following extra properties:
* `__isProxy` - A boolean indicating that this is a reactive proxy
* `__binding` - The binding instance
Parameters
Name Type Description
prefix string The prefix to bind to
object object The object to wrap
options object Options for the binding
options.shallow boolean Whether to only wrap the top-level object
options.render boolean|function Disables rendering if set to false, or replaces with a custom render function
options.propagate boolean Whether to propagate changes to parent bindings (eg. if user.data.name changes, user.data and user also update)
options.extends boolean Fallback object to use when the key is not found
options.collapseValue boolean Whether to collapse references to the object itself to its _value property (eg. if "test" is {_value: 5} and we bind {{ test }}, 5 will be used instead of the object)
Returns
- Proxy: The reactive proxy object

valueRef

valueRef()
Creates a shallow reactive reference object with a single "value" property.
Parameters
Name Type Description
prefix string The prefix to bind to
value * The initial value of the reference
Returns
- object: The reference object with a "value" property
Examples
const countRef = reactive.valueRef("count", 0);
countRef.value = 5;

// In HTML: {{ countRef }} <!-- This will update to 5 -->
