LS.Resize

LS.Resize

A flexible and robust resizer library for LS.
Adds resize bars to any side or corner of an element.
Automatically adjusts to absolute/relative positioning and works with touch events as well.
Note: ls.css is required.

LS.Resize.set

Adds a resize handle to a target element. It can be called multiple times on the same element to change sides or options (upsert).

Adds a resize handle to a target element. It can be called multiple times on the same element to change sides or options (upsert).

Certain options (anchor, size) can also be set per specific handle with CSS (eg. element > .ls-resize-handle.ls-top { --ls-resize-handle-size: 8px; --ls-resize-anchor: 0.5 })

Defined in js/resize.js:70
LS.Resize.set(target, options)

Parameters

NameTypeRequiredDefaultDescription
target*Yes- The target element to resize.
options*Yes- The options for the resize handle.

Sides and corners can be specified in four ways:
- As a CSS-style array of booleans/numbers (e.g. { sides: [true, false, true, false] }), clockwise
- As an array of strings (e.g. { sides: ["top", "bottom"] })
- As keys { top: true, right: false, bottom: true, left: false }
- All at once { sides: "all" }

Other options:
options.styledbooleanNotrue- Whether to apply default visual styles, eg. highlight on hover. Otherwise only functional styles will apply.
options.sidesboolean|Array|stringNotrue- Which sides to add handles to. Can be a boolean (true = all sides), an array of booleans/numbers [top, right, bottom, left], or an array of strings ["top", "right", "bottom", "left"].
options.cornersboolean|Array|stringNofalse- Which corners to add handles to. Can be a boolean (true = all corners), an array of booleans/numbers [top-left, top-right, bottom-right, bottom-left], or an array of strings ["top-left", "top-right", "bottom-right", "bottom-left"].
options.topbooleanNo- Whether to add a handle to the top side.
options.rightbooleanNo- Whether to add a handle to the right side.
options.bottombooleanNo- Whether to add a handle to the bottom side.
options.leftbooleanNo- Whether to add a handle to the left side.
options.topLeftbooleanNo- Whether to add a handle to the top-left corner.
options.topRightbooleanNo- Whether to add a handle to the top-right corner.
options.bottomRightbooleanNo- Whether to add a handle to the bottom-right corner.
options.bottomLeftbooleanNo- Whether to add a handle to the bottom-left corner.
options.handleSizenumber|stringNo- Size of the side handles (thickness).
options.cornerSizenumber|stringNo- Size of the corner handles (square size).
options.anchorstringNo0.5- Whether handles should be outside, inside, or in the center of the edge. (0 = inside, 0.5 = center, 1 = outside)
options.cursorsbooleanNotrue- Whether to set the cursor when dragging (this does not effect the CSS cursor style of the handle itself).
options.boundsCursorsbooleanNotrue- Set the cursor to a single direction when dragging past min/max.
options.snapAreanumberNo40- Pixel threshold for snapping.
options.snapCollapsebooleanNofalse- If true, shrinking within snapArea snaps to collapsed height/width.
options.snapExpandbooleanNofalse- If true, expanding within snapArea of parent height snaps to 100% height/width.
options.snapVerticalbooleanNofalse- If true, enables snapping vertically.
options.snapHorizontalbooleanNofalse- If true, enables snapping horizontally.
options.minWidthnumberNo20- Minimum width of the target element (overrides CSS min-width value).
options.minHeightnumberNo20- Minimum height of the target element (overrides CSS min-height value).
options.maxWidthnumberNonull- Maximum width of the target element (overrides CSS max-width value).
options.maxHeightnumberNonull- Maximum height of the target element (overrides CSS max-height value).
options.boundarystring|objectNonull- Boundary to constrain resizing. Can be "viewport" or a rect object {x, y, width, height}.
options.storestringNonull- Key name for storage. If set, updates will be persistent and saved into a storage object.
options.storeStringifybooleanNotrue- Whether to stringify the stored data.
options.storageobjectNonull- Custom storage object (must implement getItem/setItem). Default is localStorage.
options.translatebooleanNo- Use translate3d instead of left/top

Returns

An object with the registered handles
Events on handle:
- resize: Emitted when the element is resized with the new width, height, and state.
- start: Emitted when resizing starts.
- end: Emitted when resizing ends.

Examples

LS.Resize.set(element, {
    sides: ["top", "bottom"],
    corners: [1, 0, 1, 0]
});
