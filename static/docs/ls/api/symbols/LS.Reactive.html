LS.Reactive.wrap

Wraps an object with a reactive proxy.
The proxy will get the following extra properties:
* `__isProxy` - A boolean indicating that this is a reactive proxy
* `__binding` - The binding instance

Defined in js/reactive.js:597
LS.Reactive.wrap(prefix, object, options = {})

Parameters

NameTypeDescription
prefixstringThe prefix to bind to
objectobjectThe object to wrap
optionsobjectOptions for the binding
options.shallowbooleanWhether to only wrap the top-level object
options.renderboolean|functionDisables rendering if set to false, or replaces with a custom render function
options.propagatebooleanWhether to propagate changes to parent bindings (eg. if user.data.name changes, user.data and user also update)
options.extendsbooleanFallback object to use when the key is not found
options.collapseValuebooleanWhether to collapse references to the object itself to its _value property (eg. if "test" is {_value: 5} and we bind {{ test }}, 5 will be used instead of the object)

Returns
