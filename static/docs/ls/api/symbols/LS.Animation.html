LS.Animation.animate

Creates a new animation for the target object.
Note: Animations are short-lived and removed once completed.

Defined in js/animation2.js:325
LS.Animation.animate(target, properties = {}, options = {}, group = null)

Parameters

NameTypeRequiredDescription
targetobject|Array|NodeList|string|functionYesThe target object to animate. Optionally a selector, function or array.
propertiesobject|ArrayYesThe properties to animate, either as an object or entries ([[property, [from, to]]]).
properties.propertyobjectNoThe property to animate. Value can be a number or instance of LS.Color. Optionally you can set an initial value as [from, to].
optionsobjectYesThe animation options
options.easingstringYesThe easing function to use (linear, ease-in, ease-out, ease-in-out)
options.durationnumberYesThe duration of the animation in milliseconds
options.cutGroupnumberYesThe cut group ID. Animating with this ID stops any other animations with the same ID.
groupstringYesThe animation group
